# Manual de Operaciones de Fishing
## Versión: 1.0 | Fecha: 08-Feb-2026 | API RP 54, RP 7-1[web:27][web:32]

### 1. Preparación
- Tag fish depth ±1%.
- Free point indicator (stretch calc).
- BOP rams configurados (API 53).

### 2. Herramientas (OEM Certificadas)
| Tool     | Uso                  | SWL (klb) | Normas |
|----------|----------------------|-----------|--------|
| Overshot | External catch      | 500      | API 7L |
| Jar      | Impacto             | 750      | 7G-2  |
| Magnet   | Junk metálico       | 200      | -     |

### 3. Secuencia (IADC)
1. Circ @200% rate; rotate.
2. Jar ±stroke (up/down per stuck dir).
3. Back-off si >12hrs (e-line/chemical cut).
4. POOH con safety factor 1.5 tensile.[web:32]

### 4. Seguridad (API RP 54)
- JSA pre-job; PPE nivel 3.
- Max pull 80% yield DP.

### 5. Contingencias
- Si pack-off: High-vis pill.
- Junk: Impression block.
